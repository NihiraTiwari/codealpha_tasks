# CodeAlpha Cybersecurity Internship Portfolio

This repository contains the complete set of tasks completed during my cybersecurity internship. Each project focuses on core cloud security, encryption practices, and defense structures.

## 📁 Project Index

### 🔹 [Task 1: Secure Cloud Storage with Redundancy System](./Task-1_Redundancy_System/)
* **Core Mechanisms:** SHA-256 file hashing, duplicate block detection, mock decentralized storage nodes.
* **Objective:** Mitigate data footprint leaks while preserving data integrity across multi-zone backups.

### 🔹 [Task 2: SQL Injection Defense Application](./Task-2_SQL_Injection_Defense/)
* **Core Mechanisms:** SQLite query parameterization, CBC-mode AES-256 database field encryption.
* **Objective:** Neutralize input-bypassing vectors and securely isolate credential storage tables.

### 🔹 [Task 3: Cloud-Based Secure Bus Pass Tracker](./Task-3_Bus_Pass_System/)
* **Core Mechanisms:** Salted SHA-256 digital signature tracking token tokens, automated token validation.
* **Objective:** Prevent data tampering and counterfeit replication vectors on transit networks.
